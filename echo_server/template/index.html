Golang Websocket Demonstration

Simple demonstration of a web browser client communicating with a Golang websocket server. The client connects using the browser's built-in Websocket Javascript interface. After a successfully establishing the connection, the client sends messages across the link at a regular interval. The Golang server receives the messages and immediately retransmits them back. Messages sent from the client to server are displayed below in the left panel. The right panel displays messages as they are received from the server along with the rount-trip time in seconds.

Complete source available on Github.

      Golang on the server: 
                package main

          import (
                  "net/http"
                  "github.com/inyotech/golang-ws/ws"
          )

          func wsHandler(ch chan *ws.Frame) {

		  defer close(ch)

		  for {
			  frame, ws_ok := <-ch
				  if !ws_ok {
					  return
				  }
				  ch<-frame
			  }
		  }
          }

          func main() {
                  ...
                  http.Handle("/echo_service", ws.WsHandlerFunc(options.wsHandler))

                  http.ListenAndServe(":80", nil)

          }
